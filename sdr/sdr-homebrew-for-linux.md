# SDR: Homebrew for Linux

The decision is that I'm going to use Homebrew for Linux.


## Details

- Homebrew has worked perfectly for me to express custom formulas, like my OpenJDK ones.
- Homebrew has worked perfectly for switching between SDK versions, as my own variant of SDKMAN, `nvm`, and similar tools.
- Homebrew for Linux is decent enough, and it even has formulas for the latest versions of Nushell.
- I'm absolutely interested in Nix, but I need to make my configuration and installation story very clear first, and only then consider a migration to Nix, relying heavily on the LLM to help with that migration.
