# my-software

My personal software: dot files, utility programs, setup instructions, and more.


## Overview

This is a monorepo of software programs, software packages, software configuration files, and software-related writing. It spans multiple programming languages. It is not confined to a single orchestration tool or organizational structure.

Here is an incomplete index:

- [mac-os/](mac-os/): My personal instructions for configuring macOS the way I like it and installing the tools I use.
- [bash/](bash/): My Bash config and notes about Bash auto-completion (I always forget how to set this up!).
- [homebrew/](homebrew/): Like many Mac users, I use Homebrew to manage packages.
- [java/](java/): Java and Kotlin code that supports my personal workflows.
- [javascript/](javascript/): JavaScript code that supports my personal workflows.
- [jetbrains/](jetbrains/): My configuration for JetBrains IDEs (e.g. Intellij and Android Studio).
- [karabiner/](karabiner/): My configuration for [Karabiner-Elements](https://github.com/pqrs-org/Karabiner-Elements).
- [nushell/](nushell/): Nushell code and config that supports my personal workflows.
- [sdr/](sdr/): This directory contains my *Software Decision Records* (SDRs).
- [vscode/](vscode/): My Visual Studio Code user configuration.
- [starship/](starship/)
- [backstage/](backstage/): This is the "backstage": extra README content, repository TODO lists, and more.
